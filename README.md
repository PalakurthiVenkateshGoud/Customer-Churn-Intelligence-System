## Project Overview

This project analyzes customer churn behavior using machine learning and risk segmentation techniques. Multiple classification models were evaluated to predict customer attrition, identify churn drivers, and generate actionable retention strategies.

## Key Features

- Customer Churn Analysis
- Multiple ML Model Comparison
- Risk Tier Segmentation
- Feature Importance Analysis
- Customer Behavior Analytics
- Interactive Streamlit Dashboard
- Business Recommendations
